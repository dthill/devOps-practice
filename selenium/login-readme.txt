# Automate a Web Application.
The task for this assessment was to automate a login form.
AS a starting point the website provided by selenium to test the Selenium Webdriver was used.
The simple application searches for two input fields and inputs a mock username and password.
Once completed the form is submitted by clicking on the submit button.
To check if the made actions were successful the page is then search for a success message.
